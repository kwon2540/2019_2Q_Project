# Buylog

## 概要

 - 買い物リストアプリ
 - アカウント認証はFirebaseを使ったAppleLoginを使う
 - データの管理はFirebaseのDatabaseを使って管理する
 - デザイン資料はSketchを使う

## アーキテクチャー

 - MVVM
 
## 使うライブラリー
#### Cocoapods and SPM
 - SwiftLint
 - RxSwift
 - Firebase系
 - Charts
 
## 開発環境
|   |version| 
|---|---|
|Xcode|12.1| 
|Swift|5| 
|iOS|12.2 以上| 
|支援端末|iPhone X 以上|
